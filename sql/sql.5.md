# SQL Database Query

## challenge #5
Buatlah query SQL untuk menampilkan top 3 karyawan dengan gaji tertinggi. Tampilkan `first_name`, `last_name`, `department`, dan `salary`. Jika ada karyawan dengan gaji yang sama, urutkan berdasarkan `last_name` ascending.

## database schema
```sql
-- Menggunakan tabel employees yang sama seperti soal #1
```

## expected output
```
first_name | last_name | department  | salary
-----------|-----------|-------------|--------
Charlie    | Wilson    | Engineering | 70000.00
Alice      | Brown     | Sales       | 65000.00
Jane       | Smith     | Marketing   | 60000.00
```

## keyword
SELECT, ORDER BY, LIMIT, DESC, ASC

Easy
