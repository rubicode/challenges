# While Loop

## challenge #4
Buatlah perulangan while yang mencari posisi pertama huruf 'a' dalam string "programming". Loop harus berhenti segera setelah menemukan huruf 'a' pertama.

## driver code
```javascript
let kata = "programming";
let i = 0;
let ditemukan = false;
// Tulis kode while loop di sini untuk mencari huruf 'a'
if (ditemukan) {
  console.log("Huruf 'a' ditemukan di posisi: " + i);
} else {
  console.log("Huruf 'a' tidak ditemukan");
}
```

## output
```javascript
Huruf 'a' ditemukan di posisi: 5
```

## keyword
While, search algorithm, early termination, string processing, flag variable
