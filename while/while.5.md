# While Loop

## challenge #5
Buatlah perulangan while yang terus membagi angka dengan 2 sampai hasilnya kurang dari 1. Tampilkan setiap langkah pembagian. Mulai dengan angka 100.

## driver code
```javascript
let angka = 100;
let langkah = 0;
// Tulis kode while loop di sini untuk pembagian berulang
console.log("Proses selesai setelah " + langkah + " langkah");
```

## output
```javascript
Langkah 1: 100 / 2 = 50
Langkah 2: 50 / 2 = 25
Langkah 3: 25 / 2 = 12.5
Langkah 4: 12.5 / 2 = 6.25
Langkah 5: 6.25 / 2 = 3.125
Langkah 6: 3.125 / 2 = 1.5625
Langkah 7: 1.5625 / 2 = 0.78125
Proses selesai setelah 7 langkah
```

## keyword
While, mathematical operations, condition-based termination, step counter
