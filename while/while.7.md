# While Loop

## challenge #7
Buatlah countdown timer menggunakan while loop yang berhenti ketika mencapai 0 ATAU ketika user menekan tombol (simulasi dengan variabel). Timer mulai dari 10 detik.

## driver code
```javascript
let waktu = 10;
let tombolDitekan = false;
// Simulasi: tombol akan ditekan pada detik ke-7
// Tulis kode while loop di sini untuk countdown timer
if (tombolDitekan) {
  console.log("Timer dihentikan oleh user pada detik ke-" + waktu);
} else {
  console.log("Countdown selesai!");
}
```

## output
```javascript
Countdown: 10
Countdown: 9
Countdown: 8
Countdown: 7
Timer dihentikan oleh user pada detik ke-7
```

## keyword
While, timer logic, multiple exit conditions, OR operator, simulation
