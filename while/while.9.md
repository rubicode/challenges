# While Loop

## challenge #9
Buatlah perulangan while untuk menghitung berapa kali angka harus dikuadratkan sampai hasilnya lebih besar dari 1000. Mulai dengan angka 2.

## driver code
```javascript
let angka = 2;
let iterasi = 0;
// Tulis kode while loop di sini untuk pengkuadratan berulang
console.log("Angka " + 2 + " harus dikuadratkan " + iterasi + " kali untuk melebihi 1000");
console.log("Hasil akhir: " + angka);
```

## output
```javascript
Iterasi 1: 2^2 = 4
Iterasi 2: 4^2 = 16
Iterasi 3: 16^2 = 256  
Iterasi 4: 256^2 = 65536
Angka 2 harus dikuadratkan 4 kali untuk melebihi 1000
Hasil akhir: 65536
```

## keyword
While, exponential growth, mathematical sequence, threshold condition
