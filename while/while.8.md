# While Loop

## challenge #8
Buatlah perulangan while yang menghapus semua spasi dari awal string "   hello world   " sampai tidak ada spasi di awal lagi. Tampilkan proses penghapusan.

## driver code
```javascript
let teks = "   hello world   ";
let langkah = 0;
// Tulis kode while loop di sini untuk menghapus spasi awal
console.log("Hasil akhir: '" + teks + "'");
console.log("Total spasi yang dihapus: " + langkah);
```

## output
```javascript
Langkah 1: '  hello world   '
Langkah 2: ' hello world   '  
Langkah 3: 'hello world   '
Hasil akhir: 'hello world   '
Total spasi yang dihapus: 3
```

## keyword
While, string manipulation, charAt, substring, condition-based processing
