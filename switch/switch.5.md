# Switch Statement

## challenge #5
Buatlah kalkulator sederhana menggunakan switch statement yang menangani operator (+, -, *, /, %). Tampilkan hasil operasi atau pesan error jika operator tidak valid.

## driver code
```javascript
let angka1 = 15;
let angka2 = 4;
let operator = "%";
// Tulis kode switch statement di sini
```

## output
```javascript
15 % 4 = 3
```
