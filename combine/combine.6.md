# Combine - Medium Level

## challenge #6
Buatlah function `analyzeNumbers` yang menerima array angka dan mengembalikan objek berisi count angka positif, negatif, dan nol. Gunakan for loop dan if-else.

## driver code
```javascript
// Tulis function analyzeNumbers di sini

let result = analyzeNumbers([1, -2, 0, 3, -1, 0, 5]);
console.log(result);
```

## output
```javascript
{ positive: 3, negative: 2, zero: 2 }
```

## keyword
Function, for loop, if-else, array processing, object return

## difficulty
Medium
