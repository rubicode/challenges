# Combine - Easy Level

## challenge #3
Buatlah function `getDayName` yang menerima parameter `dayNumber` (1-7) dan mengembalikan nama hari menggunakan switch statement. Jika nomor tidak valid, return "Invalid day".

## driver code
```javascript
// Tulis function getDayName di sini

console.log(getDayName(1));
console.log(getDayName(5));
console.log(getDayName(8));
```

## output
```javascript
Senin
Jumat
Invalid day
```

## keyword
Function, switch statement, day mapping, return value

## difficulty
Easy
