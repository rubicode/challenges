# Combine - Medium Level

## challenge #10
Buatlah function `sumUntilLimit` yang menggunakan for loop untuk menjumlahkan angka 1,2,3... dan while loop untuk berhenti ketika jumlah mencapai atau melebihi limit.

## driver code
```javascript
// Tulis function sumUntilLimit di sini

console.log(sumUntilLimit(10));
console.log(sumUntilLimit(20));
```

## output
```javascript
{ sum: 10, count: 4 }
{ sum: 21, count: 6 }
```

## keyword
Function, for loop, while loop, sum calculation

## difficulty
Medium
