# Combine - Hard Level

## challenge #19
Buatlah function `ticTacToe` yang mengimplementasikan logika game tic-tac-toe menggunakan nested for untuk board checking, if untuk win conditions, dan switch untuk player moves.

## driver code
```javascript
// Tulis function ticTacToe di sini

let board = [
  ['X', 'O', 'X'],
  ['O', 'X', 'O'], 
  ['X', 'O', 'X']
];
console.log(ticTacToe(board));
```

## output
```javascript
{
  winner: "X",
  winType: "diagonal",
  gameOver: true,
  moves: 9
}
```

## keyword
Function, nested for loops, if conditions, switch statement, game logic

## difficulty
Hard
