# Combine - Hard Level

## challenge #15
Buatlah function `findPattern` yang mencari pola dalam matrix 2D menggunakan nested for loops dan if statements untuk matching.

## driver code
```javascript
// Tulis function findPattern di sini

let matrix = [
  [1, 2, 3],
  [4, 5, 6],
  [7, 8, 9]
];
console.log(findPattern(matrix, [5, 6]));
console.log(findPattern(matrix, [2, 3, 6]));
```

## output
```javascript
{ found: true, position: [1, 1] }
{ found: false, position: null }
```

## keyword
Function, nested for loops, if statements, pattern matching

## difficulty
Hard
