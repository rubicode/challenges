# Combine - Easy Level

## challenge #2
Buatlah function `printNumbers` yang menerima parameter `n` dan mencetak semua angka dari 1 sampai n menggunakan for loop. Function tidak mengembalikan nilai, hanya mencetak ke console.

## driver code
```javascript
// Tulis function printNumbers di sini

printNumbers(5);
console.log("---");
printNumbers(3);
```

## output
```javascript
1
2
3
4
5
---
1
2
3
```

## keyword
Function, for loop, console.log, iteration

## difficulty
Easy
