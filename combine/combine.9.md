# Combine - Medium Level

## challenge #9
Buatlah function `processMenu` yang menerima pilihan menu (1-3) dan jumlah item. Gunakan switch untuk menu dan if untuk validasi jumlah (min 1). Return total harga atau pesan error.

## driver code
```javascript
// Tulis function processMenu di sini

console.log(processMenu(1, 2)); // Nasi Goreng Rp15000
console.log(processMenu(2, 0)); // Invalid quantity
console.log(processMenu(4, 1)); // Invalid menu
```

## output
```javascript
Rp 30000
Invalid quantity
Invalid menu
```

## keyword
Function, switch statement, if validation, menu processing

Medium
