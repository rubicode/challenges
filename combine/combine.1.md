# Combine - Easy Level

## challenge #1
Buatlah function `checkAge` yang menerima parameter `umur` dan mengembalikan status berdasarkan umur. Gunakan if-else untuk menentukan: "Anak-anak" (< 13), "Remaja" (13-17), "Dewasa" (18-59), "Lansia" (>= 60).

## driver code
```javascript
// Tulis function checkAge di sini

console.log(checkAge(10));
console.log(checkAge(15));
console.log(checkAge(25));
console.log(checkAge(65));
```

## output
```javascript
Anak-anak
Remaja
Dewasa
Lansia
```

## keyword
Function, if-else, age classification, return value

## difficulty
Easy
