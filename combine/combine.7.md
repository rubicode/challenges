# Combine - Medium Level

## challenge #7
Buatlah function `calculateGrades` yang menerima array nilai dan menghitung berapa banyak setiap grade (A,B,C,D,E) menggunakan for loop dan switch.

- A : 86 - 100
- B : 76 - 85
- C : 66 - 75
- D : 56 - 65
- E : 0 - 55

## driver code
```javascript
// Tulis function calculateGrades di sini

let grades = calculateGrades([85, 92, 67, 78, 45, 88]);
console.log(grades);
```

## output
```javascript
{ A: 2, B: 2, C: 1, D: 0, E: 1 }
```

## keyword
Function, for loop, switch statement, grade counting

## difficulty
Medium
