# If Statement

## challenge #7
Buatlah program yang menentukan grade nilai berdasarkan skor. Gunakan if-else if-else statement:
- Jika >= 90: "A"
- Jika >= 80: "B" 
- Jika >= 70: "C"
- Jika >= 60: "D"
- Selainnya: "E"

## driver code
```javascript
let skor = 85;
// Tulis kode if-else if-else statement di sini
```

## output
```javascript
Skor: 85
Grade: B
```
