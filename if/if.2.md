# If Statement

## challenge #2
Buatlah program yang mengecek apakah seseorang sudah dewasa (umur >= 18). Jika sudah dewasa, tampilkan pesan "Anda sudah dewasa" dan "Anda bisa memilih dalam pemilu". Gunakan hanya statement if (tanpa else).

## driver code
```javascript
let umur = 20;
// Tulis kode if statement di sini
console.log("Terima kasih");
```

## output
```javascript
Anda sudah dewasa
Anda bisa memilih dalam pemilu
Terima kasih
```

## keyword
If statement, age validation, multiple actions, adult verification
