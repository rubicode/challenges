Buatlah program yang menentukan kategori BMI (Body Mass Index). Gunakan if-else if-else statement:
- Jika < 18.5: "Underweight"
- Jika < 25: "Normal"
- Jika < 30: "Overweight" 
- Selainnya: "Obese"

## driver code
```javascript
let berat = 70; // kg
let tinggi = 1.75; // meter
// Hitung BMI terlebih dahulu
// Tulis kode if-else if-else statement di sini
```

## output
```javascript
BMI: 22.86
Kategori: Normal
```
