# If Statement

## challenge #6
Buatlah program yang menentukan diskon berdasarkan jumlah pembelian. Gunakan if-else if statement (tanpa else di akhir):
- Jika >= 500000: diskon 20%
- Jika >= 200000: diskon 10%
- Jika >= 100000: diskon 5%

## driver code
```javascript
let totalBelanja = 350000;
// Tulis kode if-else if statement di sini
```

## output
```javascript
Total belanja: Rp 350000
Diskon: 10%
Total bayar: Rp 315000
```

## keyword
If-else if statement, discount calculation, shopping system, percentage calculation
