# If Statement

## challenge #9
Buatlah program sistem parkir yang menghitung tarif berdasarkan jenis kendaraan dan durasi. Gunakan if-else if-else statement:
- Motor: Rp 2000/jam
- Mobil: Rp 5000/jam  
- Truk: Rp 10000/jam
- Selainnya: "Kendaraan tidak diizinkan"

## driver code
```javascript
let jenisKendaraan = "mobil";
let durasi = 3; // jam
// Tulis kode if-else if-else statement di sini
```

## output
```javascript
Jenis kendaraan: mobil
Durasi parkir: 3 jam
Tarif per jam: Rp 5000
Total bayar: Rp 15000
```
