# If Statement

## challenge #3
Buatlah program yang mengecek apakah sebuah angka adalah genap atau ganjil. Gunakan if-else statement untuk menampilkan hasil yang sesuai.

## driver code
```javascript
let angka = 7;
// Tulis kode if-else statement di sini
```

## output
```javascript
7 adalah bilangan ganjil
```

## keyword
If-else statement, even odd check, modulo operator, binary decision
