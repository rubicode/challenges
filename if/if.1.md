# If Statement

## challenge #1
Buatlah program yang mengecek apakah sebuah angka adalah bilangan positif. Jika positif, tampilkan pesan "Angka positif ditemukan!". Gunakan hanya statement if (tanpa else).

## driver code
```javascript
let angka = 15;
// Tulis kode if statement di sini
console.log("Program selesai");
```

## output
```javascript
Angka positif ditemukan!
Program selesai
```

## keyword
If statement, positive number, conditional execution, simple condition
