# If Statement

## challenge #10
Buatlah program kalkulator sederhana yang melakukan operasi berdasarkan operator. Gunakan if-else if-else statement:
- Jika operator "+": lakukan penjumlahan
- Jika operator "-": lakukan pengurangan
- Jika operator "*": lakukan perkalian
- Jika operator "/": lakukan pembagian (cek pembagi tidak nol)
- Selainnya: "Operator tidak valid"

## driver code
```javascript
let angka1 = 10;
let angka2 = 3;
let operator = "/";
// Tulis kode if-else if-else statement di sini
```

## output
```javascript
10 / 3 = 3.33
```

If-else if-else statement, calculator, mathematical operations, division by zero, operator validation
