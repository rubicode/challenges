```javascript

// Tulis kode do-while loop di sini untuk validasi password
```

```javascript
// Jika user input: abc, 12345, abc123
// Maka output:
Buat password Anda: abc
Password tidak valid! Harus minimal 6 karakter dan mengandung angka.
Buat password Anda: 12345
Password tidak valid! Harus minimal 6 karakter dan mengandung angka.
```
