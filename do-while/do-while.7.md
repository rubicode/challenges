# Do-While Loop

## challenge #7
Buatlah program simulasi mesin cuci yang harus menjalankan siklus pencucian minimal sekali. Setelah selesai, tanya apakah ingin menambah siklus pembilasan. Tampilkan progress setiap tahap.

## driver code
```javascript
let tahap = ["Mengisi air", "Mencuci", "Membilas", "Mengeringkan"];
let tambahBilas;
let siklusSelesai = 0;
// Tulis kode do-while loop di sini untuk simulasi mesin cuci
console.log("Mesin cuci selesai! Total siklus: " + siklusSelesai);
```
