# Do-While Loop

## challenge #1
Buatlah program yang meminta user memasukkan angka dan menampilkan kuadratnya. Program harus berjalan minimal sekali, kemudian tanya apakah user ingin melanjutkan. Gunakan "y" untuk ya dan "n" untuk tidak.

## driver code
```javascript
let angka;
let lanjut;
// Tulis kode do-while loop di sini
console.log("Terima kasih telah menggunakan program!");
```

## output
```javascript
// Jika user input: 5, y, 3, n
// Maka output:
Masukkan angka: 5
Kuadrat dari 5 adalah: 25
Apakah ingin melanjutkan? (y/n): y
Masukkan angka: 3
Kuadrat dari 3 adalah: 9
Apakah ingin melanjutkan? (y/n): n
Terima kasih telah menggunakan program!
```

## keyword
Do-while, post-condition, at least once execution, user continuation
