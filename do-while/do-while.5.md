# Do-While Loop

## challenge #5
Buatlah program yang menghasilkan angka random (1-100) dan menampilkannya, kemudian cek apakah angka tersebut genap. Jika ganjil, ulangi sampai mendapat angka genap. Program harus generate minimal satu angka.

## driver code
```javascript
let angkaRandom;
let percobaan = 0;
// Tulis kode do-while loop di sini untuk generate angka genap
console.log("Angka genap ditemukan: " + angkaRandom + " dalam " + percobaan + " percobaan");
```

## output
```javascript
// Jika random: 37, 83, 24
// Maka output:
Percobaan 1: 37 (ganjil, coba lagi)
Percobaan 2: 83 (ganjil, coba lagi)
Percobaan 3: 24 (genap, selesai!)
Angka genap ditemukan: 24 dalam 3 percobaan
```

## keyword
Do-while, random generation, post-condition check, generate-then-validate
