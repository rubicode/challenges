# Array Processing

## challenge #2
Buatlah function `analyzeElements` yang menerima parameter `arr` dan mengembalikan array berisi objek dengan informasi setiap elemen: nilai, index, dan tipe data.

## driver code
```javascript
// Tulis function analyzeElements di sini

console.log(analyzeElements([10, "hello", true]));
console.log(analyzeElements([1, 2, 3]));
```

## output
```javascript
[
  { value: 10, index: 0, type: "number" },
  { value: "hello", index: 1, type: "string" },
  { value: true, index: 2, type: "boolean" }
]
[
  { value: 1, index: 0, type: "number" },
  { value: 2, index: 1, type: "number" },
  { value: 3, index: 2, type: "number" }
]
```

## keyword
Array indexing, typeof operator, for loop, array of objects

## difficulty
Easy
