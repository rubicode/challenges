# For Loop

## challenge #11
Buatlah perulangan for yang dapat menampilkan semua elemen dalam array buah-buahan berikut beserta indeksnya.

## driver code
```javascript
let buah = ["apel", "jeruk", "pisang", "mangga", "anggur"];
// Tulis kode for loop di sini untuk menampilkan indeks dan nilai array
```

## output
```javascript
Indeks 0: apel
Indeks 1: jeruk
Indeks 2: pisang
Indeks 3: mangga
Indeks 4: anggur
```

## keyword
Loop, for, array, index, length, iteration
