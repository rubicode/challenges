## driver code
```javascript
let kata = "programming";
let jumlahVokal = 0;
// Tulis kode for loop di sini untuk menghitung vokal
console.log("Jumlah vokal: " + jumlahVokal);
```

## output
```javascript
Jumlah vokal: 3
```
