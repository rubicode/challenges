# For Loop

## challenge #9
Buatlah perulangan for yang dapat menghitung faktorial dari angka 5 (5! = 5 x 4 x 3 x 2 x 1).

## driver code
```javascript
let angka = 5;
let faktorial = 1;
// Tulis kode for loop di sini untuk menghitung faktorial
console.log("Faktorial dari " + angka + " adalah: " + faktorial);
```

## output
```javascript
Faktorial dari 5 adalah: 120
```

## keyword
Loop, for, factorial, multiplication, accumulator
