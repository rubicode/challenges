# For Loop

## challenge #10
Buatlah perulangan for yang dapat menampilkan 8 angka pertama dari deret Fibonacci (0, 1, 1, 2, 3, 5, 8, 13).

## driver code
```javascript
let a = 0, b = 1;
console.log(a); // angka pertama
console.log(b); // angka kedua
// Tulis kode for loop di sini untuk 6 angka berikutnya
```

## output
```javascript
0
1
1
2
3
5
8
13
```

## keyword
Loop, for, fibonacci, sequence, addition, variables
