# Function

## challenge #2
Buatlah function bernama `getCurrentTime` yang tidak memiliki parameter dan mengembalikan (return) waktu saat ini dalam format string "HH:MM:SS". Gunakan objek Date untuk mendapatkan waktu.

## driver code
```javascript
// Tulis function getCurrentTime di sini

// Panggil function dan simpan hasilnya
let waktu = getCurrentTime();
console.log("Waktu sekarang: " + waktu);
```

## output
```javascript
Waktu sekarang: 14:30:25
```

## keyword
Function declaration, no parameters, return value, Date object, string formatting
