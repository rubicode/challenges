# Function

## challenge #3
Buatlah function bernama `greetUser` yang menerima satu parameter `nama` dan menampilkan pesan sapaan personal menggunakan console.log. Function tidak perlu return value.

## driver code
```javascript
// Tulis function greetUser di sini

// Panggil function dengan berbagai nama
greetUser("Budi");
greetUser("Sari");
```

## output
```javascript
Halo Budi, selamat datang!
Halo Sari, selamat datang!
```

## keyword
Function declaration, single parameter, string concatenation, console.log
