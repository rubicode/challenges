# Function

## challenge #6
Buatlah function bernama `calculateRectangleArea` yang menerima dua parameter: `panjang` dan `lebar`. Function mengembalikan (return) luas persegi panjang.

## driver code
```javascript
// Tulis function calculateRectangleArea di sini

// Panggil function dan hitung luas
let luas1 = calculateRectangleArea(10, 5);
let luas2 = calculateRectangleArea(7, 3);
console.log("Luas persegi panjang (10x5): " + luas1);
console.log("Luas persegi panjang (7x3): " + luas2);
```

## output
```javascript
Luas persegi panjang (10x5): 50
Luas persegi panjang (7x3): 21
```

## keyword
Function declaration, multiple parameters, return value, area calculation
