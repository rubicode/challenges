# Function

## challenge #5
Buatlah function bernama `displayUserInfo` yang menerima tiga parameter: `nama`, `umur`, dan `kota`. Function menampilkan informasi lengkap user menggunakan console.log tanpa return value.

## driver code
```javascript
// Tulis function displayUserInfo di sini

// Panggil function dengan data berbeda
displayUserInfo("Ahmad", 25, "Jakarta");
displayUserInfo("Lisa", 30, "Bandung");
```

## output
```javascript
=== Info User ===
Nama: Ahmad
Umur: 25 tahun
Kota: Jakarta

=== Info User ===
Nama: Lisa
Umur: 30 tahun
Kota: Bandung
```

## keyword
Function declaration, multiple parameters, console.log, data display
