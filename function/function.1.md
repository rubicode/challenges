# Function

## challenge #1
Buatlah function bernama `sayHello` yang tidak memiliki parameter dan hanya menampilkan pesan "Hello, World!" menggunakan console.log. Function tidak perlu return value.

## driver code
```javascript
// Tulis function sayHello di sini

// Panggil function
sayHello();
```

## output
```javascript
Hello, World!
```

## keyword
Function declaration, no parameters, console.log, void function
