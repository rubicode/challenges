# String Processing

## challenge #3
Buatlah function `convertCase` yang menerima parameter `text` dan `mode`. Mode bisa "upper", "lower", atau "title". Return string sesuai mode yang dipilih.

## driver code
```javascript
// Tulis function convertCase di sini

console.log(convertCase("hello world", "upper"));
console.log(convertCase("HELLO WORLD", "lower"));
console.log(convertCase("hello world", "title"));
```

## output
```javascript
HELLO WORLD
hello world
Hello World
```

## keyword
toUpperCase, toLowerCase, string manipulation, title case

## difficulty
Easy
