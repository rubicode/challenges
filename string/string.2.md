# String Processing

## challenge #2
Buatlah function `analyzeCharacters` yang menerima parameter `text` dan mengembalikan array berisi setiap karakter beserta posisinya dalam string.

## driver code
```javascript
// Tulis function analyzeCharacters di sini

console.log(analyzeCharacters("Hi"));
console.log(analyzeCharacters("Code"));
```

## output
```javascript
[
  { char: "H", position: 0 },
  { char: "i", position: 1 }
]
[
  { char: "C", position: 0 },
  { char: "o", position: 1 },
  { char: "d", position: 2 },
  { char: "e", position: 3 }
]
```

## keyword
String length, charAt, for loop, array of objects

Easy
