# String Processing

## challenge #1
Buatlah function `getStringInfo` yang menerima parameter `text` dan mengembalikan objek berisi informasi dasar string: panjang karakter, karakter pertama, dan karakter terakhir.

## driver code
```javascript
// Tulis function getStringInfo di sini

console.log(getStringInfo("Hello"));
console.log(getStringInfo("JavaScript"));
console.log(getStringInfo("A"));
```

## output
```javascript
{ length: 5, first: "H", last: "o" }
{ length: 10, first: "J", last: "t" }
{ length: 1, first: "A", last: "A" }
```

## keyword
String length, charAt, string indexing, basic string properties

## difficulty
Easy
