# String Processing

## challenge #6
Buatlah function `processWords` yang menerima parameter `sentence` dan mengembalikan objek berisi array kata-kata, jumlah kata, dan kalimat yang direverse (urutan kata dibalik).

## driver code
```javascript
// Tulis function processWords di sini

console.log(processWords("Hello beautiful world"));
console.log(processWords("JavaScript is awesome"));
```

## output
```javascript
{
  words: ["Hello", "beautiful", "world"],
  count: 3,
  reversed: "world beautiful Hello"
}
{
  words: ["JavaScript", "is", "awesome"],
  count: 3,
  reversed: "awesome is JavaScript"
}
```

## keyword
split, join, array reverse, word processing

## difficulty
Medium
